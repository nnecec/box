# SearchBox.Table

## Usage

<SearchBox
  onSearch={values => console.log(values)}
  name="your-module-name"
>
  <SearchBox.Table
    columns={columns}
    dataSource={[]}
    pagination={{
      total: 100
    }}
  />
</SearchBox>

## SchemaForm API

| 参数    | 说明         | 类型                               | 默认值 |
| ------- | ------------ | ---------------------------------- | ------ |
| name    | field name   | `string`                           | -      |
| configs | tabs options | `{label: string, value: string}[]` | -      |
